# Multiplayer Photon Test (Unity)

Este é um projeto simples e básico criado para testar e demonstrar funcionalidades básicas de multiplayer usando Photon PUN 2 na Unity.
(Atualmente está imcompleto e serve apenas para testar conhecimentos básicos de Photon)

## Funcionalidades

- Conexão a salas usando Photon
- Instanciação de jogadores sincronizada
- Cada jogador tem uma cor única, sincronizada via InstantiationData
- Movimento controlado apenas localmente (não afeta jogadores remotos)

## Tecnologias

- Unity 2022.3.62f1
- Photon PUN 2 (Free Plan)
